Apa itu HTML?

HTML adalah bahasa markup untuk mempuat struktur dan konten halaman web. HTML digunakan untuk menentukan elemen elemen seperti teks,gambar,link dan lain lain pada halaman web.

Dengan HTML, kamu dapat membuat halaman web yang memiliki struktur yang jelas dan dapat di akses oleh browser web. HTML adalah fondasi dasar dari pengembangan WEB dan digunakan bersama dengan CSS dan JavaScript untuk membuat halaman web yang interaktif dan menarik.

Struktur dari HTML itu sendiri terdiri dari tag-tag yang digunakan untuk membungkus konten dan memberikan makna pada elemen elemen halaman web

Manfaat atau kelebihan dari HTML itu sendiri adalah relatif mudah untuk dipelajari,HTML dapat digunakan untuk membuat berbagai jenis halaman web, dan HTML adalah standar untuk membuat halaman web
